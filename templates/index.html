Projeto 03 — Smmarts (Android / WebView)

Entrega para cliente com app empacotado a partir do template Sngine: identidade visual no launcher e splash, URL de produção e artefatos de release prontos para a Play Console.

Voltar aos projetos ↑

Problema

Publicar a experiência web Sngine como app Android com marca consistente, assinatura release e comportamento estável ao abrir o domínio de produção em WebView.

Solução

Configuração do projeto (nome, applicationId para a Play), geração de mipmaps e splash a partir dos PNGs do cliente, ajuste de rede quando o servidor redireciona HTTPS para HTTP (usesCleartextTraffic e URL base alinhada ao redirect), e pipeline de build (assembleRelease / bundleRelease) com keystore de release.

Resultado

Pacote pronto para testes (APK) e envio à loja (AAB), com documentação de entrega para o cliente instalar e submeter versões seguintes com a mesma assinatura.
